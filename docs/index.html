Sttxt_v2 - SMS to txt

Basic app without interface to save received SMS messages to txt file for MRE platform (including Nokia S30+). Tested on Nokia 225 with display resolution 240x320. If exists "e:\Sttxt_v2.txt" text file all SMS messages saved to this txt file, if not SMS mesages saved each in individual autonamed txt files (changing app name change default text file name). For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top/ https://vxpatch.luxferre.top. Application file - "Sttxt_v2.vxp".
